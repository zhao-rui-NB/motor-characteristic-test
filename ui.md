## 軟體功能

### 主頁
```
高雄科技大學電機系
馬達特性測試系統

單相馬達
三相馬達
exit結束
```

### 單相
```
單相馬達測試
1.無載測試
2.堵轉測試
3.鐵損分離測試
4.頻率變動測試
5.馬達參數輸入
6.測試結果查詢
7.載入馬達參數
8.綜合測試
9.測試結果列印
10.回上頁
```
